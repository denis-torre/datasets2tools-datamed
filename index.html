Overview

What is a Canned Analysis?

A Canned Analysis is a pre-run analysis of a biomedical dataset by computational tool. It is stored on a website and can be easily accessed by any user.

Where are they indexed?

All information about Canned Analyses, relevant datasets, and tools is currently indexed in a database managed by the Ma'ayan Lab at the Icahn School of Medicine at Mount Sinai.

Where are they hosted?

Canned Analysis results are hosted on webpages which are created and maintained by the developers of the analysis.

Key Features

Dataset

The first key feature of a Canned Analysis is a biomedical dataset. It must be identified by a unique accession, and must be hosted online on a repository.

Tool

The second key feature is a computational tool. It must be able to perform an analysis on the specified datasets, and exists on an independent website.

URL

The third key feature is the Canned Analysis URL. It is a hard-coded link to a webpage containing the results of the analysis run by the tool on the dataset.

Annotations

Canned Analyses can be annotated by adding a brief text description, and any number of customizable metadata tags or keywords.

Properties

Findable

Canned analyses relevant to the desired data can be easily found by using the browsing interfaces. They are available in dataset search result pages and dataset landing pages.
 

Accessible

Canned analyses can be accessed by clicking on the links provided in the browsing interfaces. They are displayed as small icons containing the tool's logo.

Sharable

Canned analyses can be easily shared by copy-pasting their URLs, or by embedding them as icons in websites. The interface provides easy access to both these solutions.
Developed by the Ma'ayan Lab at the Icahn School of Medicine at Mount Sinai.
